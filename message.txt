        <meta itemprop="position" content="1">
        <h2 class="section-title"><i class="fas fa-upload"></i> 1. Upload & unzip hvm.zip</h2>
        <div itemprop="itemListElement" itemscope itemtype="https://schema.org/HowToDirection">
          <div class="instruction-card">
            <p><i class="fas fa-folder-open" style="color: #2b6c9e; width: 1.6rem;"></i> Use <strong>SFTP/FTP</strong> to transfer <code>hvm.zip</code> into <strong>/root/</strong> directory of your VPS.</p>
            <div class="command-block">
              <i># after upload, ssh into your server and run:</i><br>
              cd /root<br>
              unzip hvm.zip<br>
              ls -la hvm/   <span style="color: #7aa8d9;"># verify contents</span>
            </div>
            <p><i class="fas fa-check-circle" style="color: #22c55e;"></i> If <code>unzip</code> not installed: <span style="background: #eef5ff; padding: 0.2rem 0.8rem; border-radius: 30px;">apt install unzip -y</span></p>
          </div>
        </div>
      </section>

      <!-- STEP 2 -->
      <section id="step2" class="section-block" itemprop="step" itemscope itemtype="https://schema.org/HowToStep">
        <meta itemprop="position" content="2">
        <h2 class="section-title"><i class="fas fa-cog"></i> 2. Python/pip & system config</h2>
        <div itemprop="itemListElement" itemscope itemtype="https://schema.org/HowToDirection">
          <div class="instruction-card">
            <p>Install pip and allow <strong>break-system-packages</strong> (for global env).</p>
            <div class="command-block">
              apt update && apt install python3-pip -y<br>
              mkdir -p ~/.config/pip<br>
              echo -e "[global]\nbreak-system-packages = true" > ~/.config/pip/pip.conf
            </div>
            <p style="margin-top: 0.7rem;"><i class="fa-regular fa-note-sticky"></i> This avoids externally-managed-environment errors.</p>
          </div>
        </div>
      </section>

      <!-- STEP 3 -->
      <section id="step3" class="section-block" itemprop="step" itemscope itemtype="https://schema.org/HowToStep">
        <meta itemprop="position" content="3">
        <h2 class="section-title"><i class="fas fa-download"></i> 3. Install dependencies</h2>
        <div itemprop="itemListElement" itemscope itemtype="https://schema.org/HowToDirection">
          <div class="instruction-card">
            <p>Move into the <code>hvm</code> directory and install requirements.</p>
            <div class="command-block">
              cd /root/hvm<br>
              pip install -r requirements.txt
            </div>
            <div class="tip-note" style="margin: 1rem 0 0;">
              <i class="fas fa-clock"></i> This may take 2–3 minutes – about 130+ packages (see step 6 for full list).
            </div>
          </div>
        </div>
      </section>

      <!-- STEP 4: systemd service file -->
      <section id="step4" class="section-block" itemprop="step" itemscope itemtype="https://schema.org/HowToStep">
        <meta itemprop="position" content="4">
        <h2 class="section-title"><i class="fas fa-file-alt"></i> 4. Create systemd service</h2>
        <div itemprop="itemListElement" itemscope itemtype="https://schema.org/HowToDirection">
          <div class="instruction-card">
            <p>Create the service unit file using <code>nano</code> (or vim).</p>
            <div class="command-block">
              sudo nano /etc/systemd/system/hvm.service
            </div>
            <div class="file-edit">
[Unit]
Description=HVM Panel (Discord Bot)
After=network.target

[Service]
User=root
WorkingDirectory=/root/hvm
ExecStart=/usr/bin/python3 /root/hvm/hvm.py
Restart=always
RestartSec=5
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=multi-user.target
            </div>
            <div class="btn-copy" onclick="copyServiceContent()"><i class="far fa-clipboard"></i> Copy service block</div>
          </div>
        </div>
      </section>

      <!-- STEP 5: reload & start -->
      <section id="step5" class="section-block" itemprop="step" itemscope itemtype="https://schema.org/HowToStep">
        <meta itemprop="position" content="5">
        <h2 class="section-title"><i class="fas fa-play-circle"></i> 5. Reload daemon & start</h2>
        <div itemprop="itemListElement" itemscope itemtype="https://schema.org/HowToDirection">
          <div class="instruction-card">
            <p>After saving the service file:</p>
            <div class="command-block">
              sudo systemctl daemon-reload<br>
              sudo systemctl restart hvm<br>
              sudo systemctl status hvm   <span style="color: #9bc0f0;"># check if active (green)</span>
            </div>
            <p><i class="fa-regular fa-circle-check" style="color: #15803d;"></i> Enable on boot: <code>sudo systemctl enable hvm</code></p>
          </div>
        </div>
      </section>
